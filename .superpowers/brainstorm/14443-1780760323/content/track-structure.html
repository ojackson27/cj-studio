Section 2: Track Playbook Structure

Every track playbook has the same 5 sections — the content changes, the shape doesn't. This is what Claude reads when it picks up a project.

Playbook Anatomy (all 4 tracks)

Same structure, different content per track

1
Track Brief Template
A fill-in-the-blanks form: client name, business description, goals, colour preferences, reference sites (from Q5), photography status. This becomes the brief handed to Claude.
2
Design Direction Rules
The non-negotiable aesthetic constraints for this track. Animation style, layout philosophy, photography treatment, typography mood. Claude never deviates from these.
3
Tool & Component List
Exact packages and APIs for this track. No guessing. e.g. Hospitality always uses Lenis + Framer Motion reveals + Google Maps. Claude pulls from this list only.
4
ARCHITECTURE.md Template
A pre-filled architecture file for this track — tech stack locked, component structure defined, brand placeholders clearly marked. Claude fills in the client-specific blanks.
5
Human Review Gates
Exactly where you or Josh must stop and approve before Claude continues. Defined per track — a hospitality site needs a photography approval gate that a trades site doesn't.

What Changes Per Track

Highlights of how the 4 playbooks differ

🍽 Hospitality
Animation: Slow scroll reveals, parallax hero, appetite-driven imagery
Layout: Full-bleed photography, minimal text, emotion-first
Key section: Menu display + location/hours + atmosphere gallery
Review gate: Photography approval before any build starts
🏗 Trades & Construction
Animation: Clean entrance animations, no gimmicks — trust-first
Layout: Project gallery hero, reviews prominent, CTA above the fold
Key section: Quote form + project gallery + accreditations/reviews
Review gate: CTA copy and quote form fields approved by client
💼 Professional Services
Animation: Subtle, purposeful — nothing that distracts from credibility
Layout: Typography-led, clear service hierarchy, conversion funnel
Key section: Services + credentials + Cal.com booking embed
Review gate: Services copy and pricing structure approved before build
🎨 Creative & Portfolio
Animation: GSAP ScrollTrigger, custom cursor, experimental transitions
Layout: Showstopper hero, work first, everything else secondary
Key section: Full portfolio showcase + artist statement + contact
Review gate: Visual direction moodboard approved before any code
